Accédez rapidement aux derniers titres

Notre application est simple d'utilisation : accédez en un tapotement à toutes les catégories d'articles. Les bannières renforcent le contenu et permettent de connaître le contenu de l'article sans même lire le titre.

Nous avons optimisé les performances de l'application pour un chargement rapide et une utilisation fluide. Le contenu essentiel est gardé en cache pour le retrouver efficacement dès le chargement suivant.

Lisez confortablement nos articles

La présentation des articles est claire et va à l'essentiel : pas de popups agaçantes, de boutons de réseaux sociaux... Concentrez-vous sur la lecture de votre article dans une page optimisée uniquement pour cela.
En plus, nos articles sont écrits avec amour pour vous proposer le meilleur contenu <3

En clair... comme en foncé

Vous aimez les journées ensoleillées d'été ? Nous disposons d'un thème clair. Vous préférez les cavernes sombres ? Nous avons également un thème sombre. Choisissez l'apparence qui vous plaît parmis nos deux magnifiques thèmes, et lisez vos articles confortablement.

Si vous le souhaitez, le thème peut s'adapter à celui de votre système. Fini les bidouillages indépendants dans chaque application pour ne pas vous détruire les yeux à l'ouverture.

Pas de publicités. Pas de traçage. C'est aussi simple que ça.

Étant nous-mêmes utilisateurs de nombreuses applications de lecture sur nos téléphones, nous avons souhaité reproduire leurs avantages sans porter leurs inconvénients. C'est pour cela qu'aucune publicité n'est affichée dans l'application. De plus, nous ne récoltons aucune donnée, pas même anonymes. La statistique est loin d'être notre domaine de spécialité, croyez-nous.

Vous souhaitez le vérifier par vous-mêmes ? Aucun problème, l'application est distribuée en open-source sous licence GNU GPL v3. Par ailleurs, n'hésitez-pas à contribuer à son développement!

Installer une application vous incomode?

Pas de panique. Grâce à notre application web progressive (PWA) directement installée sur le site becauseofprog.fr, vous pouvez ajouter le site sur votre écran d'accueil pour un accès facile à tous les articles. Vous ne profiterez cependant pas de futures fonctionnalités, telles que les notifications à la publication de nouveaux contenus.

  • Sur Chrome

    Si vous êtes sur mobile / tablette : Allez dans les 3 petits points en haut à droite de l'écran et touchez "Ajouter à l'écran d'accueil".

    Si vous êtes sur PC : Allez dans les 3 petits points en haut à droite de l'écran, allez dans "Plus d'outils" et cliquez sur "Ajouter au bureau". N'oubliez pas de cocher la case "Ouvrir dans une fenêtre".
    Retrouvez-y même sur chrome://apps

  • Sur le navigateur android 4.2+

    Ce navigateur ne supporte pas les applications web !

  • Sur Dolphin

    Ce navigateur ne supporte pas encore les fonctionnalités les plus récentes, utilisez plutôt Chrome ou Opera. Allez dans vos favoris et appuyez longuement sur BecauseOfProg. Sur le menu qui s'affiche cliquez sur "Ajouter un raccourci sur l'écran d'accueil"

  • Sur Firefox

    Ce navigateur ne supporte pas encore les fonctionnalités les plus récentes, utilisez plutôt Chrome ou Opera. Appuyez longuement sur l'URL et cliquez sur "Ajouter à l'écran d'accueil"

  • Sur Opera Mini

    Ce navigateur ne supporte pas les applications web. Toutefois, vous pouvez prendre la version complète d'Opera (instructions juste en dessous).

  • Sur Opera

    Allez dans les 3 petits points en haut à droite de l'écran et touchez "Ecran d'accueil" tout en bas.

  • Sur iOS (avec Safari)

    Appuyez sur le bouton du milieu en bas de l'écran, et appuyez sur "Ajouter à l'écran d'accueil".
